# Lab 0

Lab 0 是人工智能引论课程的入门实验，主要用于熟悉 Python 基础语法、NumPy 的基本操作，以及后续 Pacman 实验中会用到的简单地图表示方式。

本实验的主要代码在 `exercise_0.py` 中，内容包括：

- Python 列表、字典、排序和类的基础练习；
- 使用 NumPy 创建数组、进行矩阵和条件替换操作；
- 用随机采样方法估计圆周率；
- 读取 Pacman 地图文件，并在结构化表示和文本表示之间转换。

目录中的 `example.lay` 是示例地图文件，`autograder.py` 用于本地检查实验完成情况。

运行方式：

```powershell
python autograder.py
```

这个实验更偏向课程热身，目的是为后续搜索、机器学习和 Pacman 智能体实验打好 Python 与数据结构基础。
